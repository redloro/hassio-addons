# Xantech Multi-Zone Audio Amplifiers and Controllers

# Supported Hardware

- MRAUDIO8X8 / MRAUDIO8X8m
- MRC88 / MRC88m
- MX88 / MX88a / MX88ai / MX88vi

# Unsupported

The following do not seem to support the Xantech multi-zone RS232 API:

- MRAUDIO4X4
- MRC44 / MRC44CTL
